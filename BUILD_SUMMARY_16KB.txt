================================================================================
INSTALLATION & FIRST RUN
================================================================================

Step 1: Connect Android Device
  - Enable USB Debugging (Developer Settings)
  - Connect via USB cable

Step 2: Install APK
  adb install app/build/outputs/apk/debug/app-debug.apk

Step 3: Launch App
  - Open "MedGuide" on the device
  - Or: adb shell am start -n com.runanywhere.kotlin_starter_example/.MainActivity

Step 4: Navigate to Chat
  - From Home screen, tap "Chat - LLM" button

Step 5: Load Models (First Time Only)
  - Tap "Load Models" button
  - Wait for download (~610MB on first run)
  - This only happens ONCE - models are cached

Step 6: Verify Voice Features
  - Mic button should appear next to Send button
  - STT and TTS models will be pre-loaded

Step 7: Test Voice Input
  - Tap mic button
  - Speak a medical emergency query
  - App transcribes speech and sends to AI
  - AI responds with first-aid guidance
  - Response plays aloud automatically

Step 8: Go Offline!
  - Turn off Wi-Fi and mobile data
  - App works 100% offline
  - All AI features function normally
